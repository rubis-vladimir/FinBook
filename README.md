  Используя ###FinBook### вы можете эффективно **вести личные финансы** (учет доходов/расходов)
  
  Программа 
  + с возможностью поиска по названию или категории, 
  + построения наглядной диаграммы, 
  + сохранению данных на устройстве, 
  + а также выбора темной/светлой темы.

```Swift
addConstraint(NSLayoutConstraint(
    item: button1,
    attribute: .Right,
    relatedBy: .Equal,
    toItem: button2,
    attribute: .Left,
    multiplier: 1.0,
    constant: -12.0
))
```

### Примененные технологии:
+ Язык программирования SWIFT
+ Архитектура MVC
+ Фреймворк UIKit
+ CoreData - сохранение транзакций на устройстве
+ UserDefaults - сохранение состояния цветовой темы
+ JSON - парсинг данных для карточек разработчиков
+ Делегирование, Singletone

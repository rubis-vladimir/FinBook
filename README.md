  ### Программа поддерживает:
  + возможность поиска по названию или категории; 
  + построение наглядной диаграммы;
  + сохранение данных на устройстве; 
  + выбор темной/светлой темы.

  ### Примененные технологии:
  + SWIFT 5.0
  + Архитектура MVC 
  + Фраемворк UIKit  
  + CoreData 
  + UserDefaults 
  + JSON-парсинг данных 
  + Делегирование
  + Singletone
  + Generics
